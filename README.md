# Setup Instructions

Follow these steps to get the project up and running.

## 1. Clone the Repository

```bash
git clone https://gitlab.com/username/project-name.git
cd project-name
```

## 2. Install the dependencies

```bash
npm install
```

## 3. Deploy with Docker

```
docker compose build
docker compose up
```
